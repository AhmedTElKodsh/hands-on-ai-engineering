## 🗂️ File Organization Principles

### 1. Single Canonical Path

**Rule:** Only one active curriculum version

**Implementation:** `LAYER1-FINAL/` is canonical

**Benefit:** No confusion about which to use

### 2. Archive Old Versions

**Rule:** Never delete, always archive

**Implementation:** `_ARCHIVE_2026-03/`

**Benefit:** Historical reference preserved

### 3. Separate Planning from Content

**Rule:** Planning docs separate from curriculum

**Implementation:** `_PLANNING-2026-03/`

**Benefit:** Clear distinction between planning and teaching

### 4. Clear Naming Conventions

**Rule:** Descriptive, consistent names

**Examples:**
- `LAYER1-FINAL/` - Clear it's the final version
- `_PLANNING-2026-03/` - Underscore prefix for sorting
- `_ARCHIVE_2026-03/` - Underscore prefix for sorting

**Benefit:** Easy to find and understand
