# BMAD Master - Workflow Orchestrator

This steering file activates the BMAD (Build Me A Dream) Master agent for workflow orchestration.

## Quick Reference

**Activation**: Use `#bmad-master` in chat
**Deactivation**: Say "exit bmad mode" or "return to normal mode"

## What This Does

Loads the complete BMAD Master agent persona and workflow system from:
`#[[file:.clinerules/workflows/bmad-core-agents-bmad-master.md]]`

The full agent definition includes:

- Project context and component overview
- Orchestration protocols and execution principles
- Workflow modules (BMB, BMM, CIS)
- Code conventions and testing requirements
- Development patterns and critical rules

## Usage

When activated, the agent will:

1. Discover existing context (specs, outputs)
2. Identify appropriate workflows
3. Propose structured approach
4. Execute with phase transitions and validation
5. Document decisions and summarize results

For detailed instructions, see the referenced workflow file above.
