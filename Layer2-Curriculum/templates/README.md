# Chapter Templates

This directory contains the official chapter templates for the Hands-On AI Engineering curriculum.

## Template Version: v2.1

All new chapters must follow the MASTER-CHAPTER-TEMPLATE-V2.md structure.

## Required Sections

1. **Metadata Block**: Phase, time, difficulty, prerequisites, project thread
2. **Verification Section**: 3-5 automated test scripts with expected outputs
3. **Summary Section**: Minimum 7 bullet points, key takeaway, skills unlocked
4. **Hands-On Exercises**: Minimum 2 "Try This!" challenges

## Quality Standards

- 100% template compliance required
- All code must be runnable
- Verification scripts must pass
- Civil Engineering examples preferred
