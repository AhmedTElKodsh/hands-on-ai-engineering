## What's What?

### Modified Curriculum
- **32-week structure** with flex weeks
- **Guided learning** (TODOs, hints, checkpoints)
- **GenAI Engineer focus** (RAG, Agents, LLM Apps)
- **Teaching methodology** for AI assistants
- **Portfolio-first** (3 flagship projects)

### Original Layer 1
- **40-day structure** with daily mini-projects
- **Complete examples** and working code
- **Comprehensive coverage** of AI engineering
- **Mechanic's workflow** pedagogy
- **Action-first** approach

---

## How to Use Both

### Option 1: Modified as Primary
✅ Use modified structure (32 weeks, flex time)  
✅ Reference original for technical content  
✅ Use original examples as inspiration  

### Option 2: Original as Primary
✅ Follow original 40-day plan  
✅ Add checkpoints from modified  
✅ Use teaching methodology for AI help  

### Option 3: Hybrid
✅ Modified structure + Original content  
✅ Best of both worlds  

---

## Key Differences

| Aspect | Original | Modified |
|--------|----------|----------|
| **Duration** | 40 days | 32 weeks |
| **Structure** | Daily mini-projects | Weekly themes + flagships |
| **Code** | Complete examples | Scaffolded TODOs |
| **Teaching** | Build first, understand after | Build with guidance, explain to proceed |
| **Flexibility** | Linear | Flex weeks built in |
| **Focus** | Comprehensive AI Engineer | GenAI Engineer (RAG/Agents) |
